# dotfiles

## How To Use

0. Make sure git is installed on your system

1. Clone the repository

$ git clone https://github.com/rejoice4156/dotfiles.git

2. Edit variables in .bashrc.d/variables.bashrc

  - $USER_NAME
  - $USER_EMAIL
  - $USER_SIGNINGKEY
  - $SSH_PUBKEY

3. Make init.sh executable

$ chmod +x init.sh

3. Run init.sh

$ sh init.sh

4. Reload bash or log out and log back in

$ bash exec
