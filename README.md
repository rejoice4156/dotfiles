# dotfiles

## How To Use

0. Make sure git is installed on your system

1. Clone the repository

$ git clone https://github.com/rejoice4156/dotfiles.git

2. Edit variables in .bashrc.d/variables.bashrc

  - $USER_NAME
  - $USER_EMAIL

3. Make init.sh executable

$ chmod +x init.sh

3. Run init.sh

$ sh init.sh
